Recriação da API de temperatura desenvolvido no curso de angular. Essa api foi conectada com uma API KEY do site openweathermap.

Angular versão 15.2.11
node versão 20.15.0

Funcionalidades adicionadas:
- Imagem do ceu no fundo da tela;
- Tela responsiva;
- Dois novos containers que retornam a temperatura mais fria e a mais quente das cidades listadas no json.
